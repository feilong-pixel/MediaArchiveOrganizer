# Media Archive Organizer 免责声明

## 1. 使用风险说明

本工具用于对图片和视频文件进行自动整理。
在使用过程中，程序可能会对文件执行移动或拷贝操作，并在目标目录中创建新的目录结构。

虽然本工具已经尽量按照既定规则处理文件，但在实际使用时，仍然可能因文件权限、路径错误、磁盘状态、文件占用、异常中断、时间信息异常或其他不可预见原因导致以下情况：

- 文件整理结果不符合预期
- 文件被移动到非预期目录
- 同名文件被自动重命名
- 个别文件整理失败
- 日志记录不完整
- 极端情况下出现数据丢失、覆盖或无法恢复的风险


## 2. 用户责任

使用者在运行本工具前，应自行确认以下事项：

- 源目录与目标目录填写正确
- 重要文件已经提前备份
- 磁盘空间充足
- 对 `move` 模式和 `copy` 模式的区别有明确理解
- 对程序整理后的目录结构有基本预期

如果你希望保留原始文件，请优先使用 `--mode copy`。


## 3. 关于移动模式的特别说明

当使用默认的 `move` 模式时，文件会从源目录移动到目标目录。
这意味着整理完成后，源目录中的原文件可能不再保留。

因此，在首次使用、批量处理重要资料、或不确定整理结果是否符合需求时，建议先：

- 使用少量测试文件验证效果
- 优先使用 `--mode copy`
- 检查日志内容后再进行正式处理


## 4. 关于时间识别的说明

本工具会优先读取图片的 EXIF 时间；
如果无法读取 EXIF 时间，则使用文件修改时间作为备选日期。

因此，最终整理结果依赖于文件本身的时间信息是否准确。
如果原始 EXIF 信息错误、缺失，或文件修改时间已被改变，整理后的日期目录可能与拍摄真实日期不一致。


## 5. 关于日志的说明

本工具会为每次执行生成日志文件，但日志仅作为辅助排查和结果记录用途。
日志并不构成对整理结果完整性、准确性或可恢复性的保证。

使用者应在重要任务执行后，自行检查：

- 日志中是否存在 `ERR`
- 目标目录中的文件数量是否符合预期
- 目标目录结构是否正确


## 6. 关于重复检测的说明

重复检测结果仅作为操作参考，不构成绝对判断。

- `strict` 模式仅将字节级完全一致的文件视为重复
- `phash` 模式基于视觉相似度判断，不能保证两个文件在业务、法律或归档意义上属于完全相同
- 即使程序将文件标记为重复，涉及重要资料时仍应由使用者人工复核
